# PDF Summarizer

A Python script that summarizes the content of a PDF file.

---

## 🚀 Getting Started

Welcome! This guide will help you set up your environment, install Python, and run this project on your computer—even if you've never used a terminal before.

---

## 💻 Step 1: Open Your Terminal

The terminal (or command line) is just a text-based way to talk to your computer. Here is how to open it based on your operating system:

* **Windows:** Press the **Windows Key**, type `PowerShell` or `cmd`, and press **Enter**.
* **Mac:** Press `Cmd + Space` to open Spotlight, type `Terminal`, and press **Enter**.
* **Linux:** Press `Ctrl + Alt + T` on your keyboard.

---

## 📥 Step 2: Install Python 3

This project requires **Python 3** to run. Let's check if you already have it, or install it if you don't.

### 1. Check if Python is already installed
Type the following command into your terminal and press **Enter**:

```bash
python3 --version
```

f that doesn't work, try python --version

If it returns something like Python 3.x.x: Great! Skip to Step 3.

If it says "command not found" or opens the Microsoft Store: Follow the download instructions below.

How to Download and Install Windows: 
1. Go to python.org/downloads.
2. Click the yellow button to download the latest version for Windows.
3. CRITICAL STEP: When running the installer, make sure to check the box that says "Add python.exe to PATH" before clicking Install.

Mac:
1. Go to python.org/downloads.
2. Download the macOS installer and run it.
3. Alternatively, if you use Homebrew, type: brew install python

Linux (Ubuntu/Debian):

Open your terminal and run:
```bash
sudo apt update && sudo apt install python3 python3-pip
```

## 📂 Step 3: Navigate to the Project Folder

Before you can run the project, your terminal needs to "point" to the folder where the project files are saved. We use the cd (change directory) command for this.

Download or clone this project repository to your computer.

Open your terminal.

Type cd  (with a space after it), then drag and drop your project folder from your file explorer directly into the terminal window. It will automatically fill in the path!

Press Enter.

💡 Tip: You can type ls (Mac/Linux) or dir (Windows) and press Enter to see a list of files in your current folder and make sure you are in the right place.
